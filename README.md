# ras0q.github.io

GitHub Pages site for personal feeds.

## Starred repository releases

Atom feed: https://ras0q.github.io/starred-releases.atom

The feed is generated by
[`ras0q/starred-releases-feed`](https://github.com/ras0q/starred-releases-feed)
and published from the `gh-pages` branch.

### One-time setup

1. Add repository secret `GH_PAT` with a PAT that can read your starred
   repositories and their releases.
2. Enable GitHub Pages for this repository:
   - Source: **Deploy from a branch**
   - Branch: `gh-pages` / `/ (root)`
3. Run the **Starred Releases Feed** workflow manually once from the Actions tab.

After the first successful run, `starred-releases.atom` and `state.json` appear on
`gh-pages`.
